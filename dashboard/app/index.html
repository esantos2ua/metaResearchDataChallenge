Query components

    Sensitivity: definitions of “Canadian”

    “Canadian” can be defined several defensible ways. Holding the metaresearch subject filter constant, this shows how many works qualify under each. The rest of the dashboard uses the primary definition (any author at a Canadian institution); the others are reported side by side for transparency.

    Planned (next iteration): an interactive control to switch the active definition so every chart and the network re-aggregate accordingly. For now these are reference counts.

    Live verification

    Re-run the query against OpenAlex right now and compare the live result count with the snapshot shipped in this dashboard. Small differences are expected as OpenAlex is continuously updated.

    Relevance spot-check

    Fetch a fresh random sample of works returned by this exact query, straight from OpenAlex, and judge for yourself whether they are genuinely Canadian metaresearch.
